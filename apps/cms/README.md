# Strapi CMS (`apps/cms`)

## Основные команды

Из корня монорепозитория:

```bash
pnpm --filter cms dev
pnpm --filter cms build
pnpm --filter cms start
```

Из директории `apps/cms`:

```bash
pnpm dev
pnpm build
pnpm start
```

## Дополнительно

```bash
pnpm console
pnpm deploy
pnpm upgrade:dry
pnpm upgrade
```
